# Daily Task Tracker

This is a basic task tracker website made using HTML, CSS and JavaScript.

In this project, we can add tasks, mark them as completed, edit them and also delete them.

## Features

- Add a task
- Cannot add empty task
- Maximum 15 tasks allowed
- Mark task as completed
- Edit task
- Delete task
- Tasks are saved even after refreshing the page
- Filter tasks (All / Completed / Pending)

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project
2. Open the folder
3. Open index.html in browser
